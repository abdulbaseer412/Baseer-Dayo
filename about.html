About This Project

Hello! I’m Baseer, a student of MYP 4 at The International School, Karachi. I created this quiz website as part of my IB Personal Project.

The main goal of my project is to learn web development — specifically using HTML, CSS, and JavaScript — to design a simple, user-friendly, and functional website. This website allows students to test their knowledge in different subjects in an interactive way.

My inspiration came from my deep interest in technology and design. I’ve always been curious about how websites like YouTube and Google are made, and this project gave me a chance to start learning those skills. I also believe that technology is the future, and building digital tools can help improve education for everyone.

Through this project, I learned not only how coding works but also how creativity, logic, and design come together in real projects. In the future, I plan to expand this website with more subjects, levels, and interactive features — and eventually connect it to a database for real results.

This project reflects my journey as a learner — exploring coding, design, and technology to create something meaningful for students.
